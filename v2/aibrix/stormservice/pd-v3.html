Disaggregated Architecture Overview

This deployment strictly isolates prefill and decode phases across AMD MI300X GPUs. Traffic is managed by a stateless CPU router. The single-GPU prefill node crunches context and ships the KV cache via Mooncake to a high-throughput, gang-scheduled decode cluster spanning two physical nodes.

🎭 Component Configuration
🚦 Router Layer (CPU)
Runs aibrix/sglang-router with --pd-disaggregation. Polls port 30000 via K8s service discovery to maintain active maps of worker pods.
⚡ Prefill Engine (1x MI300X)
SGLang in prefill mode. Dedicated entirely to prompt computation. Transmits state via Mooncake.
🔄 Decode Cluster (2x MI300X)
Demands podGroupSize: 2 for atomic scheduling. Rank is mapped dynamically to the StatefulSet hostnames.
Routing Layer & Service Discovery

Operating entirely on CPU, this pod acts as the traffic cop. It requires the sglang-router-sa ServiceAccount to LIST/WATCH pods. It targets explicit replicas using the role-replica-index=0 label, ensuring traffic hits the active heads of the StatefulSets.

Prefill Hardware Layer (TP=1)

Reserved strictly for 1 MI300X GPU. Instead of token generation, it mathematically constructs the KV Cache and transmits it via Mooncake. Crucial hardware optimizations are enforced via environment variables to prevent ROCm latency spikes.

⚙️ ROCm OS Optimizations
HSA_NO_SCRATCH_RECLAIM=1Stops ROCm scratch memory reclamation.
GPU_FORCE_BLIT_COPY_SIZE=6Optimizes blit copying for small tensors on MI300X.
TORCH_BLAS_PREFER_HIPBLASLT=1Prefers hipBLASLt over rocBLAS for faster GEMMs.
securityContext: IPC_LOCKLocks memory to prevent OS paging to disk.

Distributed Decode (TP=2)

Spread across 2 physical instances. Aibrix manages podGroupSize: 2 for atomic scheduling. Rank-1 uses a pure Bash /dev/tcp polling loop to wait for Rank-0, eliminating external dependencies from the base ROCm container.

🏎️ Network Sync Variables
NCCL_IB_DISABLE=1Forces standard Ethernet usage over InfiniBand.
SYS_ADMIN & SYS_PTRACECapabilities required for Mooncake memory injection.
Bash Rank DerivationParses StatefulSet index: decode-<hash>-0-<RANK>
Container: decode | Bash Logic
# Derive Node Rank from Hostname NODE_RANK=$(echo $HOSTNAME | awk -F'-' '{print $NF}') # Derive Master DNS (Rank 0) MASTER_HOSTNAME=$(echo $HOSTNAME | sed 's/-[0-9]*$/-0/') MASTER_DNS="${MASTER_HOSTNAME}.${STORM_SERVICE_NAME}.default.svc.cluster.local" MASTER_PORT=29500 # Rank-1 blocking wait using pure bash if [ "$NODE_RANK" != "0" ]; then until timeout 3 bash -c "echo > /dev/tcp/${MASTER_DNS}/${MASTER_PORT}" 2>/dev/null; do sleep 3 done fi # Launch Engine python3 -m sglang.launch_server \ --disaggregation-mode decode \ --disaggregation-transfer-backend mooncake \ --tp-size 2 --nnodes 2 \ --node-rank $NODE_RANK \ --dist-init-addr ${MASTER_DNS}:${MASTER_PORT}
CRD Architectural Skeleton

Aibrix uses the StormService Custom Resource to package the entire inference microservice into a single deployable object. By setting stateful: true, it guarantees the stable network identities needed for multi-node Tensor Parallelism.
