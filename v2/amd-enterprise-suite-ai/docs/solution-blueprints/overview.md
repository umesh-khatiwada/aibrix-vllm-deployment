# Solution Blueprints Overview

**Solution Blueprints** are end-to-end AI application templates that you can deploy as a complete package. Instead of assembling individual components, a blueprint gives you a full working application in one deployment.

---

## What's in a Blueprint?

A blueprint bundles together:

- One or more AI models (inference services)
- A frontend or API layer
- Storage and configuration
- Pre-configured connections between components

---

## When to Use Blueprints

Use blueprints when you want a **complete application** rather than just a model:

- You want a **chat application** for your team, not just a model API
- You need a **benchmarking pipeline** that runs automatically
- You're deploying a **production AI service** and want a validated starting point
