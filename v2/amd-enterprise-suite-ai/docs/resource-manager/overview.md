# AMD Resource Manager Overview

The **AMD Resource Manager** is the control plane for your infrastructure. Administrators use it to manage hardware clusters, create projects, control user access, and configure storage and secrets.

---

## What Can You Do in the Resource Manager?

| Section | Purpose |
|---|---|
| [Clusters](clusters.md) | Monitor AMD GPU hardware health and utilization |
| [Projects](projects.md) | Create team workspaces with compute quotas |
| [Storage](storage.md) | Manage object storage buckets for models and datasets |
| [Secrets](secrets.md) | Store sensitive credentials securely |
| [Users & Access](users.md) | Manage users, roles, and authentication |

---

## Dashboard

The Resource Manager **Dashboard** gives you a real-time view of:

- Total GPU utilization across all clusters
- Running vs queued workloads
- Storage consumption
- Active users and projects

---

## Who Uses the Resource Manager?

- **Infrastructure Admins** — primary users; manage everything
- **Resource Managers** — manage projects and users within their scope
- **AI Practitioners** — limited access; view their project's resource usage
